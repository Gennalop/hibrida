Acerca de Descripción

Esta aplicación utiliza Teachable Machine para reconocer imágenes y determinar el tipo de insecto que aparece en ellas. Además, proporciona información detallada sobre el insecto identificado. Dentro de la aplicación, también puedes acceder a un directorio que incluye información sobre insectos peligrosos para la agricultura, ayudándote a identificar cuáles representan una amenaza. La aplicación usa un conjunto de datos hecho por los desarrolladores y otro que se obtuvo de Kaggle.

¿Cómo usar la aplicación?
  1. Dirígete a la sección Teachable Machine.
  2. Sube la imagen del insecto que deseas identificar.
  3. Presiona el botón Identificar Insecto.
  4. La aplicación mostrará toda la información relevante sobre el insecto.
